# Wilson Galdamez Portfolio

This is a simple personal portfolio website built with HTML and CSS to showcase my projects and provide easy access to my GitHub repositories and live demos.

## Overview

The site includes a navigation bar, a homepage link, and a projects page that highlights several coding projects I have worked on. The goal of this portfolio is to present my work in a clean and organized way so employers, classmates, and other developers can quickly view what I have built.

## Features

- Simple and clean portfolio layout
- Navigation bar for easy page switching
- Dedicated projects section
- Links to GitHub repositories
- Links to live project demos
- Easy to customize and expand

## Technologies Used

- HTML
- CSS

## Projects Included

### PacMen Exercise
A JavaScript animation exercise focused on DOM manipulation, event handling, and timing functions.

- GitHub Repo: `https://github.com/Willie-Byte/PacMen.github.io`
- Live Project: `https://willie-byte.github.io/PacMen.github.io/`

### Eye Exercise
An interactive project using CSS and JavaScript to create googly eyes that follow the user's cursor.

- GitHub Repo: `https://github.com/Willie-Byte/Eye-Exercise.github.io`
- Live Project: `https://willie-byte.github.io/Eye-Exercise.github.io/`

### Real Time Bus Tracking
A map-based project that uses live MBTA data to display real-time bus locations.

- GitHub Repo: `https://github.com/Willie-Byte/RealTimeBusTracker`
- Live Project: `https://willie-byte.github.io/RealTimeBusTracker/`

### MiniDigMasters Frontend
A React frontend for an equipment sales website. This project demonstrates responsive UI design, component structure, state management, and API integration for an e-commerce style experience.

- GitHub Repo: `https://github.com/Willie-Byte/minidig-frontend`
